Language: C++

How to compile and view run times:

1. Open main.cpp in desired IDE.  

2. From the terminal, go to project folder and do the following command: 
	
	g++ -Wall program1.cpp -o make

3. Then, type in: ./make

4. You will then be prompted by the program to choose your choice of simulation.

5. You will run through the simulation via user input until you are done.

6. You will be notified when the simulation of your choice has concluded.

7. If you choose an invalid input, you will be asked for a proper one until you have done so.
